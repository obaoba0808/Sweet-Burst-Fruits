爆甜水果鋪 商用完整版網站

使用方式：
1. 解壓縮 ZIP
2. 直接打開 index.html 預覽
3. 上傳整個資料夾到主機即可上線

檔案結構：
- index.html 首頁
- products.html 所有商品
- hot.html 熱銷推薦
- gift.html 水果禮盒
- story.html 產地故事
- contact.html 聯絡我們
- checkout.html 結帳頁
- css/style.css 樣式
- js/app.js 商品、購物車、搜尋、分類
- images/ 本地圖片素材 SVG

正式收款可再串接：
- 綠界 ECPay
- 藍新 NewebPay
- LINE Pay
- n8n 訂單自動通知
- Google Sheet / Airtable / 後台訂單管理
