GitHub Pages 修正版

上傳 GitHub 時請保持這個結構：

index.html
products.html
hot.html
gift.html
story.html
contact.html
checkout.html
css/style.css
js/app.js
images/hero.svg
images/mango.svg
images/strawberry.svg
images/grape.svg
images/pineapple.svg
images/apple.svg
images/kiwi.svg
images/gift.svg
images/family.svg
images/farm.svg
images/contact.svg

破圖最常見原因：
1. 只上傳 html，沒有上傳 images 資料夾。
2. GitHub Pages 大小寫敏感，images 不能寫成 Images。
3. 檔案沒有放在 repo 根目錄。
4. 路徑使用 /images/，在專案子路徑會破圖；本版已改成 ./images/。
